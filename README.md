# DISC SHELF

3Dバイナルレコード風UIの**パーソナルアプリポータル**。  
自作WebアプリをLPレコードとして棚に並べ、タップ／クリックで起動できます。

🔗 **Live**: <https://junia2009.github.io/disc-shelf/>  
📦 **Repo**: <https://github.com/junia2009/disc-shelf>  
🏷️ **Version**: v1.2.3

---

## 主な機能

| 機能 | 説明 |
|------|------|
| **3D シリンダーカルーセル** | Three.js 製バイナルレコードが円形に並ぶ。枚数が多い場合は自動で複数段に分割 |
| **段スワイプ** | 縦スワイプ or ホイールで段を切り替え。段インジケーター（ピルドット）で現在地を表示 |
| **カテゴリワープ** | 🌀ワープボタンでセキュリティ→ゲーム→その他を循環切り替え |
| **一覧ビュー** | カテゴリ別ソート＋リアルタイム検索（名前・説明・タグ対応） |
| **長押しツールチップ** | モバイルでディスクを500ms長押し→アプリ名・説明を表示（振動フィードバック付き） |
| **起動モーダル** | ディスクをタップするとモーダルでアプリ情報を確認してから起動 |
| **PWA 対応** | iPhoneホーム画面に追加可能。Service Workerで2回起動すれば確実に最新版に更新 |
| **オーロラ背景** | CSS グラデーションアニメーション |
| **星フィールド** | 500個のマルチカラー星が個別に明滅（Additive Blending） |
| **流れ星** | 2〜6秒間隔でランダム出現 |

## ディスクの追加方法

`app.js` の `DISCS` 配列にオブジェクトを1つ追加するだけです。

```js
{
  id: 'my-new-app',              // 一意のID（英数字・ハイフン）
  name: 'アプリ名',
  description: 'アプリの説明文',
  color: '#FF9F43',              // ディスクのテーマカラー（HEX）
  url: 'https://〜',             // アプリの公開URL
  repo: 'https://github.com/〜', // GitHub リポジトリ URL
  tags: ['JS', 'tool'],          // タグ（配列）
  category: 'セキュリティ',       // 'セキュリティ' | 'ゲーム' | 'その他'
},
```

カテゴリの段数は枚数に応じて自動計算されます（最大6枚/段）。

## バージョンアップ手順

PWA は Service Worker でキャッシュされるため、更新時は以下2ファイルのバージョンを同期して上げてください。

```js
// sw.js
const CACHE = 'discshelf-v1.2.3';  // ← 上げる

// app.js
const VERSION = 'v1.2.3';          // ← 同じ番号に揃える
```

ユーザーが PWA を2回起動すれば自動で最新版に切り替わります。
